Fianl project accessibility evaluation

Role: designer

Target site: Michigan Events

Suggestion1: The site uses only color (yellow) for its emphamsis on suggested search keywords. It can add more indicators such as underline or italic to let people who cannot distinguish color easily still have the ability to get the emphasis.

Suggestion2: The main color identity Tiffany Blue has bad contrast with the dark grey background. I suggest the site to use another font color for a better color contrast or choose not to use grey to as its background color.

Suggestion3: Most of the links on the site don't have a focus state for users who use keyboard to know where they are on the sites. It can add hover or focus state to its navigation, footer and the "card" like elements to indicate there are links on it and they're clickable.

Suggestion4: There are a lot of errors regarding the tag and alt attribute. They include "alt" attribute in the tag, which is not allowed. Also, it should use aria-label for icons and imgs to avoid empty text in the link.
